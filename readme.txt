== Description ==
**افزونه بازارا**
با افزونه بازارا به راحتی میتونید کالاهاتون رو از نرم افزار به فروشگاهتون انتقال بدید.

= Compatibility =
* Woocommerce 5.x
* Wordpress 5.x

= Support =
*  [افزونه بازارا](https://www.mahaksoft.com)

= Fa =
* امکان انتخاب نام یا مبلغ و یا تعداد کالا جهت همگام سازی به انتخاب کاربر
* امکان همگام سازی اتوماتیک با انتخاب زمان همگام سازی به انتخاب کاربر
* ثبت اتوماتیک فاکتور در نرم افزار حسابداری محک بعد از پرداخت موفق در فروشگاه
* همگام سازی مشتریان سایت به نرم افزار و بالعکس
* امکان انتخاب مشتری عمومی برای فاکتور
* امکان ایجاد دسته بندی و گروه کالا به انتخاب کاربر

== Installation ==
1. Upload `bazara` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
